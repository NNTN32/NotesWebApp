# 📘 NotesWebApp

NotesWebApp là ứng dụng tạo và quản lý ghi chú, được xây dựng theo mô hình **Backend + Frontend tách biệt**.

## 🚀 Công nghệ sử dụng
### Backend (Java Spring Boot)
- Spring Boot
- Spring Data JPA / Hibernate
- PostgreSQL
- Redis / Docker
- JWT Authentication
- Maven

### Frontend (tùy dự án bạn dùng: React/Flutter/HTML-CSS-JS)
- React (hoặc Flutter / HTML-CSS-JS)
- Axios / Fetch API
- TailwindCSS / Bootstrap (nếu có)
